About

Goals

AI, 로그 분석, 메트릭 이상 탐지, 그리고 모니터링과 분석의 융합적 연구

Groups

  • LLM 기반 로그 분석 연구(LLM for Log Analysis Research)
    주제: 로그 데이터를 NLP 및 LLM으로 분석하고 요약하는 기술 연구.
    활동: 로그 요약, 원인 분석, 자연어 기반 운영 질문 응답.
  • 메트릭 이상 탐지 연구(Metrics Anomaly Detection Research)
    주제: 시계열 데이터 및 메트릭 기반 이상 탐지 알고리즘 연구.
    활동: Prometheus 메트릭 데이터 활용, 딥러닝 및 비지도 학습 적용
  • 운영 자동화 연구(AIOps Automation Research)
    주제: AIOps에서 Function Calling과 워크플로 자동화를 연구.
    활동: 이상 탐지 이후 자동 대응 시스템 설계
  • 실시간 모니터링 데이터 연구(Real-Time Monitoring and Analytics Research)
    주제: 실시간 데이터를 분석하고 비정상 상태를 빠르게 탐지하는 기술 연구.
    활동: 스트리밍 데이터 처리, 이상 탐지 및 경고 시스템 설계.
  • 모델 성능 최적화 연구(AI Model Optimization Research)
    주제: LLM과 이상 탐지 모델의 성능 향상을 위한 최적화 연구.
    활동: 모델 파인튜닝, 전이 학습, 도메인 특화 모델 개발 및 AI Model Optimization
