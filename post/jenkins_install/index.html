Jenkin 설치

2022-10-09 Jaehoon Khim

Docker 설치

  1. dnf update -y
  2. dnf config-manager –add-repo https://download.docker.com/linux/centos/docker-ce.repo
  3. dnf install docker-ce –allowerasing
  4. systemctl start docker
  5. systemctl enable docker
  6. systemctl status docker
  7. usermod -aG docker $USER

Docker Compose 설치

  1. dnf install -y curl
  2. curl -L “https://github.com/docker/compose/releases/download/v2.11.2/docker-compose-$(uname -s)-$(uname -m)” -o /usr/local/bin/docker-compose
  3. chmod +x /usr/local/bin/docker-compose
  4. docker-compose –version

Jenkins 설치

  1. mkdir ws
  2. cd ws
  3. vim docker-compose.yml
version: '3' 

services:
	jenkins:
	image: jenkins/jenkins:lts 
	container_name: jenkins 
	volumes: 
		- /var/run/docker.sock:/var/run/docker.sock 
		- /jenkins:/var/jenkins_home 
	ports: 
		- "8080:8080" 
	privileged: true 
	user: root
  1. docker-compose up -d
  2. docker ps -a
  3. docker-compose logs
