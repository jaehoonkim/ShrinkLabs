docker image 저장소 위치 변경

  • 새로운 디스크를 추가합니다.
    • 여기에서는 “/docker” 에 mount 되어 있는걸로 가정합니다.
  • “etc/docker/daemon.json” 에 다음의 내용을 추가합니다.
{
  "data-root": "/docker"
}
  • docker 서비스를 재시작합니다.
$ sudo systemctl daemon-reload
$ sudo systemctl restart docker
