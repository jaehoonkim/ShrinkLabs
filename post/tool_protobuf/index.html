protobuf 사용

설치

  • protoc 최신 버전 확인 ( https://github.com/google/protobuf/releases )
$ curl -OL https://github.com/protocolbuffers/protobuf/releases/download/v3.15.6/protoc-3.15.6-linux-x86_64.zip

$ unzip protoc-3.6.1-linux-x86\_64.zip -d protoc3

$ sudo mv protoc3/bin/\* /usr/local/bin/
$ sudo mv protoc3/include/\* /usr/local/include/

$ sudo chown $USER /usr/local/bin/protoc

$ sudo chown -R $USER /usr/local/include/google

원본: Install protobuf 3.6.1 on Ubuntu 16.04 (github.com)
